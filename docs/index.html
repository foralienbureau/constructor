Конструктор фандрайзинговых лендингов

Плагин для WordPress, с помощью которого можно создавать красивые 'развернутые' страницы для сбора пожертвований
Скачать Документация

Конструктор предназначен для быстрого создания фандрайзинговых онлайн-акций на WordPress.

Он позволяет составить из стандартных блоков страницу-лендинг, объясняющую суть акции, мотивирующую жертвователей и дающую им возможность перечислить пожертвование.

Конструктор дает возможность быстро создавать лендинги с индивидуальным оформлением (например, под яркие рекламные кампании).

Работа с Конструктором не требует сложных технических навыков. Необходимо базовое владение системой управления контентом WordPress.

Конструктор является расширением функционала плагина «Лейка», предоставляет больше возможностей для оформления страниц и интерактивные виджеты для улучшения UX сбора.
