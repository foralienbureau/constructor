Второй пример

Соберем средства на продуктовые наборы для самых незащищенных: стариков, инвалидов, малоимущих семей.

Страница лендинга

Он позволяет составить из стандартных блоков страницу-лендинг, объясняющую суть акции, мотивирующую жертвователей и дающую им возможность перечислить пожертвование. Конструктор дает возможность быстро создавать лендинги с индивидуальным оформлением (например, под яркие рекламные кампании).

Работа с Конструктором не требует сложных технических навыков. Необходимо базовое владение системой управления контентом WordPress.

Конструктор является расширением функционала плагина «Лейка», предоставляет больше возможностей для оформления страниц и интерактивные виджеты для улучшения UX сбора. Это инструмент с открытым исходным кодом.

Как использовать

Код плагина предоставляется «как есть», разработчики не осуществляют техническую поддержку и не отвечают за корректную работу модуля на отдельном сайте. Содержание и корректная работа сайта - ответственность их создателей.
